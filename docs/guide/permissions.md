# 權限管理系統

Lomall 整合 Discord 角色系統，提供精細的權限控制。

## 權限層級

| 層級 | 角色 | 功能 |
|------|------|------|
| **3** | Admin（管理員） | 完整存取：管理設定、指派工單、檢視全部 |
| **2** | Support（客服） | 管理工單、回覆對話 |
| **1** | Viewer（檢視者） | 唯讀檢視工單與審計日誌 |

## 角色辨識

Lomall 根據 Discord 角色名稱自動判斷權限：

| 角色名稱包含 | 對應權限 |
|-------------|---------|
| `admin` | Admin |
| `support` | Support |
| `staff` | Support |
| `viewer` | Viewer |

## 頻道權限

當工單建立時，Lomall 自動設定以下 Discord 頻道權限：

- **@everyone** — 拒絕存取
- **工單建立者** — 檢視、發送訊息、讀取歷史
- **管理員角色** — 檢視、發送訊息、管理頻道
- **客服角色** — 檢視、發送訊息
